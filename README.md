# ICF キャラクター機能パラメータ - ステータス画面

ICF（国際生活機能分類）をベースにしたゲーム風キャラクター機能パラメータのWebアプリケーションです。
Final Fantasy風のスタイリッシュなユーザーインターフェースで、3層階層のナビゲーションシステムを提供します。

## 🎮 プロジェクト概要

このWebアプリは、ICF（International Classification of Functioning, Disability and Health）の分類体系をゲームキャラクターの機能パラメータとして応用し、視覚的に魅力的な形で表示・探索できるツールです。

## ✨ 主な機能

### 現在実装済みの機能

1. **3層階層ナビゲーション**
   - ドメイン → カテゴリー → コード の順次選択
   - 直感的なクリック操作でナビゲーション

2. **詳細情報表示**
   - コードの解説（医学的・リハビリテーション的視点）
   - ゲームへの応用方法
   - 具体的なゲーム例

3. **FF風UIデザイン**
   - ダークブルーとゴールドのカラーテーマ
   - グラデーション背景とグロー効果
   - アニメーション効果とホバー演出

4. **レスポンシブデザイン**
   - デスクトップ・タブレット・モバイル対応
   - 画面サイズに応じたレイアウト調整

5. **統計情報表示**
   - 総ドメイン数、カテゴリー数、コード数の表示

## 📊 データ構造

### 主要なデータフィールド
- **ドメイン**: 大分類（心身機能、身体構造、活動と参加、環境因子）
- **カテゴリー**: 中分類（各ドメイン内のサブカテゴリー）
- **コード**: 小分類（具体的な機能項目）
- **解説**: 医学的・リハビリテーション的な説明
- **ゲーム応用**: ゲームでの活用方法
- **具体例**: 実際のゲームでの応用例

### 現在登録されているデータ
- **総ドメイン数**: 4つ（心身機能、身体構造、活動と参加、環境因子）
- **総カテゴリー数**: 17カテゴリー
- **総コード数**: 239項目

## 🚀 使用方法

1. Webページを開く
2. 左側のナビゲーションパネルから**ドメイン**を選択
3. 表示された**カテゴリー**一覧から希望のカテゴリーを選択
4. **コード**一覧から具体的なコードを選択
5. 右側の詳細パネルに選択したコードの詳細情報が表示される
